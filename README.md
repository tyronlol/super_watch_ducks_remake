'super watch ducks legacy' is a remake of the popular, legendary, and famous game 'watch ducks legacy'.
To run the game, open the console and type 'curl -sSL https://bootstrap.pypa.io/get-pip.py -o get-pip.py
python get-pip.py', then 'pip install ursina', then download Python from 'https://www.python.org/downloads/', and after that, launch the game!
